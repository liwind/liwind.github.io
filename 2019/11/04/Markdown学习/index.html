Markdown学习

发表于 2019-11-04 更新于 2019-11-21 分类于 工具学习 阅读次数： Valine：

1 简介

Markdown是一种纯文本格式的标记语言，通过简单的标记语法，他可以使普通文本内容具有一定的格式。特点是易读，易写，易更改。Markdown也可以理解为将以 MARKDOWN语法编写的语言转换成HTML内容的工具。

Markdown支持的HTML元素，不在Markdown涵盖范围之内的标签，都可以直接在文档里面用HTML撰写。

4 特殊字符

4.1 空格

多个空格子表示一个空格。

使用标签：普通空格&nbsp;，半角空格&ensp;，全角空格&emsp;

示例：句子1 句子2 句子3&nbsp;&nbsp;&nbsp;句子4

效果：句子1 句子2 句子3   句子4
